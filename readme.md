# Shopping List

A simple, user-friendly shopping list application where users can manage their shopping items efficiently.

## Features

- **Add Items**: Quickly add items to your shopping list.
- **Edit Items**: Modify any item in the list with ease.
- **Remove Individual Items**: Delete specific items from the list.
- **Clear All**: Clear the entire shopping list with one click.
- **Filter Items**: Filter through the list to find specific items using a search input.
